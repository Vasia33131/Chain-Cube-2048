О проекте
Жанр: Puzzle / Match-3 / 2048-лайк
Движок: Unity
Язык: C#
Платформа: WebGL / PC / Mobile

Управление
Отпустить / подтвердить — слить цепочку в один куб следующего уровня

Или (в зависимости от реализации):

Свайп (пальцем или мышкой) — проводи линию через одинаковые кубы

Механика цепочек
Соединяй 2 одинаковых куба в ряд (по горизонтали, вертикали или ломаной линией)

Цепочка превращается в один куб следующего уровня (2→4→8→16→32→...→2048)
